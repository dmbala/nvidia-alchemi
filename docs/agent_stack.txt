Legend
------
  →    data flow
  ╔═╗  runs inside a Singularity container
  [  ] orchestrated stage (one or more SLURM jobs)

Who does what
-------------
  Agent    decides WHEN        (submit, retry, trigger, triage)
  HT       computes GROUND TRUTH   (expensive, per molecule)
  AL       decides WHICH MOLECULES to verify next (cheap, at scale)
