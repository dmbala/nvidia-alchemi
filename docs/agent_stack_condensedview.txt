  Container stack (each extends the previous):                                                
                                                         
  alchemi.sif          → base (PyTorch + CUDA + ALCHEMI toolkit + warp)                       
      ↓ extends                                                                               
  alchemi_ht.sif       → + AIMNet2, RDKit, GPU4PySCF, PySCF, ASE, dftd3 shim
      ↓ extends                                                                               
  alchemi_clip.sif     → + transformers (ChemBERTa), xgboost, SchNet (pure PyTorch)
                                                                                    

                                                                                              
  Separation of concerns:                                
  - Agent decides when (submit, retry, trigger, stop).                                        
  - HT computes ground truth (expensive, automatic per molecule).                             
  - AL decides which molecules are worth the expensive HT evaluation next (cheap prediction +
  UCB ranking).                                                                               
                                                                                              
  The agent connects the two by ticking the state machines and triaging anomalies — it never  
  touches the DFT parameters or the ML hyperparameters, which are policy frozen in the SLURM  
  scripts.
